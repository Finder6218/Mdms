# 模具管理系统

功能主要包含：   
1、基本信息：客户信息、供商信息   
2、订单管理：订单明细,客户付款、客户款统计、付供商款、供商款统计   
3、统计查询：订单对账,材料采购,加工单对账、排程计划、生产明细工单 

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe(或Hadp.exe)"文件即可完整运行对应的信息管理系统。
